# MERN Stack Project

## Structure
- `client/`: Frontend (React + Vite + Tailwind)
- `server/`: Backend (Node + Express + MongoDB)

## Setup

### Server
1. `cd server`
2. `npm install`
3. Create `.env` file with `MONGO_URI` and `PORT`.
4. `npm run dev`

### Client
1. `cd client`
2. `npm install`
3. `npm run dev`
